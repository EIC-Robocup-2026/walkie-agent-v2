## What this task wires
- **Model:** `anthropic/claude-sonnet-4.5`; human + scene perception on so any
  capability is available to demo.
- **Prompt** (`prompt.md`): a placeholder demo script to fill in. Until then the
  robot acts as a general assistant and narrates via `speak`.

Good things to show off: CLIP scene memory ("where did I last see X"), human
recognition / re-ID, gesture understanding (C7).
