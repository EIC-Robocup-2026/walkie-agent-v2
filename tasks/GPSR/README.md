# GPSR — General Purpose Service Robot Challenge (§5.3)

**Status: 🟢 ready-leaning** — planning/nav/perception/speech supported; grasps partial.

> The operator issues **three** commands (generated by the official command
> generator, then LLM-rephrased). The robot plans and executes them, then returns
> to the Instruction Point. Partial scoring applies. Max time 7:00.

```bash
./tasks/GPSR/run.sh            # start
./tasks/GPSR/run.sh fresh      # wipe scene+object DBs, then start
```

## What this task wires
- **Model:** `anthropic/claude-opus-4.5` (stronger decomposition for compound
  commands) — see `config.toml`.
- **Prompt** (`prompt.md`): take 3 commands (consecutive or interleaved for the
  bonus), `write_todos` plan, resolve references via Database/Vision/Human agents,
  answer questions by `speak`, return to the Instruction Point.
- **Perception:** 3 s live snapshot interval; scene memory on.

## Readiness notes
- ✅ Understand command / plan / navigate / perceive / report / answer questions.
- 🟡 Commands needing pick/place/pour → manipulation partial; prompt does the
  reachable parts and requests assistance for grasps (penalised, still scores).

## Overriding at launch
```bash
WALKIE_MODEL=anthropic/claude-sonnet-4.5 ./tasks/GPSR/run.sh   # cheaper run
DISABLE_LISTENING=1 ./tasks/GPSR/run.sh                        # type commands at a TTY
```
